# 開発ロードマップ

## プロジェクト概要

**目標**: 外国人技能実習生向けの日本語学習アプリ「Nihongo」を開発し、現場で即使える日本語フレーズを習得できるようにする

**開発期間**: 約3ヶ月（11週間）

**リリース目標**: MVP（Minimum Viable Product）としての基本機能を実装

**削除された機能**:
- 録音・発音練習機能
- 学習履歴トラッキング
- 連続学習日数（ストリーク）機能

## リスク管理

### 主要リスク

1. **音声データの準備**
   - リスク: 100フレーズ分の高品質な音声データ作成に時間がかかる
   - 対策: 初期はTTSで代替、段階的にプロの音声に置き換え

2. **デバイス互換性**
   - リスク: 様々なAndroid/iOSデバイスでの動作保証
   - 対策: 早期に複数デバイスでテスト実施

3. **パフォーマンス問題**
   - リスク: 音声ファイルの読み込みによるパフォーマンス低下
   - 対策: キャッシュ戦略の実装、最適化

4. **ストア審査**
   - リスク: App Store/Google Play の審査で却下される
   - 対策: ガイドライン遵守、十分なテスト期間確保
