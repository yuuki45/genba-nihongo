# App Store 表記ドラフト（名前・サブタイトル・キーワード）

作成: 2026-06-08 / 対象: App Store Connect の「製品情報」をローカリゼーションごとに入力

## 方針

- ホーム画面のアイコン名は **Nihongo** のまま（短くアイコン下に収まる）
- App Store の表示名・サブタイトルは**言語ごとに別設定**できる。実習生は日本のApp Storeでも
  端末がインドネシア語なら id 版の表記が表示される → **id を主ターゲットとして最適化**
- ブランドは **Genba Nihongo（現場にほんご）**。ありふれた "Nihongo" 単体との差別化になる
- App Store の文字数上限: **名前 30字 / サブタイトル 30字**（全角も半角も1字）

---

| 項目 | 案 | 文字数 |
|------|-----|:---:|
| **名前** | `Genba Nihongo: Bahasa Jepang` | 28 |

  `bahasa jepang,belajar,nihongo,JLPT,N3,N2,kerja,magang,pabrik,perawatan,kaigo,frasa,kanji,jisho`

## 補足

- 「名前」は検索順位への影響が最も大きい → ブランド＋最重要キーワード（Bahasa Jepang / 日本語）
- 「サブタイトル」は対象と中身（職種＋JLPT）を凝縮
- 名前・サブタイトル・キーワードに**同じ語を重複させない**（インデックスの無駄。例: 名前に Bahasa Jepang を入れたらキーワードからは外してよい）
- アイコン名（CFBundleDisplayName / android:label）は現状 `Nihongo` のまま変更なし
