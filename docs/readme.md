# 📘 要件定義書（Flutter版）

## プロジェクト名

**Nihongo – Indonesia Edition**

## 🧭 コンセプト

外国人技能実習生が、**日本の工場現場で「すぐ使える日本語」**を手軽に学べるモバイルアプリ。
「音声で聞く・話して練習・翻訳で理解」を一体化し、現場でのコミュニケーションストレスを減らす。

## 🎯 ターゲット

| 項目 | 内容 |
|------|------|
| 主対象 | インドネシアから来日する技能実習生 |
| レベル | 日本語能力試験 N5〜N2（初級〜中上級） |
| 使用環境 | iPhone・Android スマートフォン |
| 利用シーン | 工場・清掃・製造などの現場作業中または休憩時間 |
| 利用目的 | よく使う日本語フレーズを「聞いて」「話して」「覚える」 |
## 💡 目的

- 現場で即使える日本語フレーズを中心に構成
- 音声・翻訳機能をワンタップで利用可能
- お気に入り機能で重要なフレーズをすぐにアクセス

## ⚙️ 機能要件

| 機能カテゴリ | 機能名 | 概要 |
|------------|--------|------|
| 🏠 ホーム | 今日の3フレーズ | 日替わりで3件のフレーズを提示 |
| 📖 フレーズ一覧 | カテゴリ別・JLPTレベル別表示 | 「あいさつ」「安全」「作業」など分類で一覧表示 |
| 💬 フレーズ詳細 | 音声再生 / 翻訳表示 | |
| 🔊 音声再生 | 日本語音声（TTS） | |
| ⭐ お気に入り | 重要なフレーズを保存・一覧表示 | |
| 🔍 検索機能 | フレーズをキーワードで検索 | |
| ⚙️ 設定 | 言語（日本語/インドネシア語）切替、音声速度調整 | |
| 🚨 緊急フレーズ | 「助けてください」などをワンタップ再生（予定） | |
| 🔔 通知 | 学習リマインド通知（予定） | |
| 📡 オフライン対応 | JSONデータをローカルDB保存 | |
